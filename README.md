###Kumar Electricals Website

##Project Overview

This is a React-based corporate website for Kumar Electricals, a leading electrical engineering and turnkey solutions company in India. The website showcases the company profile, management team, services, clients, and specializations.

The project is designed to be fully responsive, working on both desktop and mobile devices, with modern navigation, a slideshow banner, and smooth UI effects.


###Features

1. Responsive Header & Navbar

  a.Logos on the top-left
  b.Navigation links for Home, Contact, and Webmail
  c.Bottom navigation for About, Management, Why Us, Specialization, Clients, and Services
  d.Hamburger menu for mobile devices

2. Slideshow

  a.Automatic slideshow cycling through multiple images
  b.Smooth transitions and responsive design

3.Content Pages

  a.Each page has a heading (h2) and detailed information (h5 or p)
  b.About
  c.Management
  d.Why Us
  e.Specialization
  f.Clients (with a client image grid)
  g.Services


4.Client Section

  a.Responsive grid of client logos
  b.Hover effect with subtle zoom
 
  

###Technologies Used

Frontend: React (Vite)

Routing: React Router DOM

Styling: CSS (custom + responsive design)

Icons: React Icons (FaHome, MdContactPhone, FaEnvelope, FaBars, FaTimes)

Version Control: Git/GitHub

###Usage

Navigate between pages using the top and bottom navigation bars

On mobile devices, click the hamburger menu to view navigation links

Click client logos to view details (if linked in future updates)

Hover over links and images to see animations
